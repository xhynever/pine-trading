# Pine Trading 策略集合

## 下跌趋势反弹做空策略

### Strategy V1 - 斜率筛选策略
基于4小时图表的反弹做空策略。在空头趋势中，当EMA140斜率大于25度时准备信号，等待价格反弹上穿EMA140时开空。策略采用分批入场和三级止盈机制，风险回报比约1:5。每个信号只执行一次，确保交易纪律。

### Strategy V2 - 跌幅筛选策略
同样基于4小时图表的反弹做空策略，但使用不同的筛选条件。检测过去一周（42根4H K线）内是否出现10%以上的跌幅，满足条件后等待价格反弹上穿EMA140时开空。相比V1策略，V2更关注实际的价格波动幅度而非斜率，适合捕捉剧烈下跌后的反弹机会。
